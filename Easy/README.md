# Easy Array Problems

This folder contains beginner-friendly array data structure practice problems, each with one or more C++ solutions.

## Folder Guide

- `check_sorted_array/`: Check if an array is sorted.
- `intersection of array/`: Intersection of two arrays (brute and optimal).
- `largest element/`: Find the largest element in an array.
- `left_rotate_array_by_d_places/`: Left rotate by D positions (brute and optimal).
- `max consecutives of ones/`: Find the maximum consecutive 1s.
- `missing number/`: Find the missing number (brute and optimal).
- `move zeros to end/`: Move zeros to the end (brute, optimal, optimized).
- `pass by value and reference/`: Pass-by-value vs pass-by-reference example.
- `remove duplicates/`: Remove duplicates from a sorted array.
- `Reverse an Array/`: Reverse the array.
- `rotate_array_by 1 index/`: Rotate array by 1 index.
- `second largest/`: Find the second largest element.
- `Single Number presence/`: Find the element that appears once.
- `union of arrays/`: Union of two arrays (brute and optimal).
- `vector storing in temp/`: Store vector elements in a temporary array.

## How To Use

- Open any folder and build/run the `main.cpp` (or `brute.cpp` / `optimal.cpp`) with your preferred C++ compiler.
- Compare brute vs optimal solutions to understand time and space trade-offs.

## Notes

- Folder names match the problem topic.
- Some problems include multiple approaches for learning purposes.
